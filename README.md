# Next.js Authentication Project

This is a simple authentication system built using **Next.js**, **React**, and **JWT (JSON Web Tokens)** for managing user authentication. The project includes both front-end and back-end functionality to handle user registration, login, and protected routes.

## Features

- User Registration
- User Login
- JWT Authentication
- Protected Routes (with session checking)
- Error Handling

## Technologies Used

- **Next.js**: A React framework for server-side rendering and static site generation.
- **JWT (JSON Web Tokens)**: For securing authentication.
- **React**: For building the user interface.
- **CSS/SCSS**: For styling.
- **Node.js**: Back-end for authentication logic.
